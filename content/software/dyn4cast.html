The package is not yet on CRAN, if it was, it would be possible to install the released version of Dyn4cast from CRAN in future. The canonical form for CRAN is:

install.packages("Dyn4cast")

The package is in the development version on GitHub but have undergone rigorous checks. So far all the functions are working perfectly well. Install it with:

remotes::install_github("JobNmadu/Dyn4cast")
